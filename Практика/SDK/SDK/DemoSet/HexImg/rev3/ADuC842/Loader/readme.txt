Образ загрузчика для ADuC831/ADuC842 HEX202-03. Для прошивки установить к себе на
машину и запустить Analog Devices MicroConverter(R) Windows Serial Downloader
v6.03 НЕ СТАРШЕ!  

При прошивке 
	-войти в меню Configuration
	-установить флаг Run/Bootload Option
        -установить номер COM порта
        -нажать OK
        -нажать DOWNLOAD
